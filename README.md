# Online Store Console Application

## Description of the Project
This Java console application is an online store system that allows users to view products, add items to a shopping cart, and check out with payment. The purpose of this application is to practice Java basics such as loops, conditionals, methods, file reading, ArrayLists, and user input.

The intended users are customers who want to browse products and make a purchase through a simple console-based shopping experience. The application loads product data from a `products.csv` file, displays inventory, allows users to search by product ID, adds selected products to a cart, calculates the total, accepts payment, and prints a receipt. :contentReference[oaicite:0]{index=0}

## User Stories
- As a user, I want to view all available products, so that I can decide what I want to buy.
- As a user, I want to search for a product by ID, so that I can quickly find a specific item.
- As a user, I want to add products to my cart, so that I can purchase multiple items.
- As a user, I want to view my shopping cart, so that I can see what I have selected.
- As a user, I want to see my total price, so that I know how much I owe.
- As a user, I want to enter a payment amount, so that I can complete my purchase.
- As a user, I want to receive a receipt, so that I can review my purchase and change.

## Setup

### Prerequisites
- IntelliJ IDEA: Make sure IntelliJ IDEA is installed.
- Java SDK: Make sure the Java SDK is installed and configured in IntelliJ.
- `products.csv`: The project requires a `products.csv` file containing product data.

### Running the Application in IntelliJ
1. Open IntelliJ IDEA.
2. Select **Open** and navigate to the project folder.
3. Wait for IntelliJ to index the files and set up the project.
4. Locate the `Store` class.
5. Make sure `products.csv` is in the correct project directory.
6. Right-click `Store.java`.
7. Select **Run 'Store.main()'**.
